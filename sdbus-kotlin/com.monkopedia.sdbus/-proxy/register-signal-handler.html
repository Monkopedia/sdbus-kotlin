registerSignalHandler

abstract fun registerSignalHandler(interfaceName: InterfaceName, signalName: SignalName, signalHandler: SignalHandler): Resource

Registers a handler for the desired signal emitted by the D-Bus object

Return

Resource handle to the registration

A signal can be subscribed to and unsubscribed from at any time during proxy lifetime. The subscription is active immediately after the call. The lifetime of the subscription is bound to the lifetime of the resource object. The subscription is unregistered by releasing the resource object.

Parameters

interfaceName

Name of an interface that the signal belongs to

signalName

Name of the signal

signalHandler

Callback that implements the body of the signal handler

Throws

Error

in case of failure
