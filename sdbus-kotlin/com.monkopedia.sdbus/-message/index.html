Message

expect sealed class Message

Message represents a D-Bus message, which can be either method MethodCall, MethodReply, Signal, or a PlainMessage.

Serialization and deserialization functions are provided for types supported by D-Bus.

You mostly don't need to work with this class directly if you use high-level APIs of Object and Proxy.
